# Тестовое задание для Node Js разработчика
В проекте есть база данных, в ней 4 таблицы:
- контракты
- рекламные блоки (Время во время телепрограммы, когда может быть показана реклама)
- рекламные ролики
- споты (кросс-таблица блоков и отправленных роликов)

Контракты - это договор, заключенный на размещение рекламы 

Необходимо создать:
1. Дописать API, в котором будет два метода GET и POST
2. Метод POST должен принимать следующие параметры в виде JSON: 
```
    {
        "contractID": <ID контракта>,
        "filmName": <Название ролика>,
        "filmDur": <Продолжительность ролика в секундах>
    }
```
После отправки данных должна создаться запись в таблице роликов (Нужно создать модель для роликов Film). После создания записи должен запуститься алгоритм расстановки роликов.
Данные для расстановки нужно будет взять из таблицы контрактов. contractID передается в POST-запросе. Из таблицы contracts по этому ключу взять поле totalRate(общая стоимость контракта) и поле cnlID (ID канала)
И на всю эту стоимость разместить ролики в блоках канала, указанного в контракте, размещать ролики пока не кончится общая стоимость РК (стоимость конкретного блока в поле rate). Результат этой расстановки записать в таблицу spots (Создать эту таблицу и модель).

3. Метод GET должен принимать один параметр ?filmID=<id ролика>, по запросу возвращаются сводные данные в формате:
```
{
    "filmID":<ID ролика>,
    "rate":<Общая набранная стоимость>,
    "dur":<Общая длительность размещенных роликов>
}
```

4. Сделать pull-request тестового задания. Инструкция: https://www.youtube.com/watch?v=SUnJ8bNCnCg
